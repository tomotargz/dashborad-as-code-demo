# dashborad-as-code-demo

## デモ内容

1. 読み取り専用の本番ダッシュボードを書き込み可能な形式でコピーして開発ダッシュボードを作成する。
2. 開発ダッシュボードに必要な変更を加える。
3. 開発ダッシュボードをexportしてリポジトリにpushする。
4. Pull Requestを作成する。Pull Requestを作成するとレビュー用の読み取り専用ダッシュボードがデプロイされ、そのURLがPull Request上にコメントされる。
5. Pull Requestをマージするとレビュー用のダッシュボードが削除され、本番用のダッシュボードが更新される。
